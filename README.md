# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Nessta interface esta sendo utilizada a versão node: '22.2.0'

## Available Scripts

Nesse programa você pode rodar:

### `npm start`

Irá rodar o servidor no modo desenvolimento.\
abra [http://localhost:3000](http://localhost:3000) para visualizar no seu Browser

A pagina ira recarregar automaticamente quando fizer alguma modificação.\
Você podera ver alguns erros pelo terminal.

Esse Programa é uma de duas partes de um teste para Dev Fullstack junior, fiz meu melhor pra deixar as telas com permissões onde usuarios comuns não conseguem acessar as telas voltadas a Admins
e que as telas pudessem fazer as requisições para a API e mostrassem os dados recebidos de maneira de facil entendimento, ainda estou me adaptando ao Frontend mas vai dar certo
